# Development Roadmap

## Phase 0: Local Frontend MVP

### Goal
Минимальная играбельная версия. Один демо-квест, работающий полностью в браузере без бэкенда.

### Features

#### Core
- [ ] Vite + React + TypeScript проект
- [ ] Tailwind CSS настройка
- [ ] Zustand store для игрового состояния
- [ ] LocalStorage persistence

#### UI Components
- [ ] Layout (двухколоночный)
- [ ] VisualPanel (левая часть — плейсхолдер)
- [ ] QuestPanel (правая часть — текст и выборы)
- [ ] PlayerStats (HP)
- [ ] Terminal (ввод JS кода)
- [ ] ActionButton (варианты ответов)

#### Quest Engine
- [ ] JSON parser для квестов
- [ ] Система локаций и переходов
- [ ] Проверка условий (conditions)
- [ ] Применение эффектов (effects)
- [ ] Валидация JS кода в терминале

#### Content
- [ ] JSON структура для квеста "Ship Repair"
- [ ] 3 локации: cabin, corridor, engine_room
- [ ] Терминал с JS-задачей

#### Polish
- [ ] Базовый тёмный sci-fi стиль
- [ ] Адаптивность для разных экранов

### Deliverable
Пользователь может открыть приложение в браузере, пройти демо-квест от начала до конца, починив систему питания корабля через написание JavaScript кода.

## Phase 1: Backend + LLM Integration

### Goal
Добавить серверную часть для безопасной работы с OpenAI API и валидации кастомных ответов.

### Features

#### Backend
- [ ] Express.js сервер
- [ ] TypeScript конфигурация
- [ ] CORS настройка
- [ ] Environment variables (.env)

#### API Endpoints
- [ ] `POST /api/validate-answer` — валидация кастомного ответа через GPT
- [ ] `GET /api/health` — проверка статуса сервера

#### LLM Integration
- [ ] OpenAI SDK интеграция
- [ ] Промпт для валидации ответов
- [ ] Обработка ошибок API

#### Frontend Updates
- [ ] Поле для кастомного ввода
- [ ] Интеграция с бэкендом
- [ ] Loading states
- [ ] Error handling

### Deliverable
Игрок может вводить свои варианты ответов, которые валидируются через GPT и влияют на игру.

### Goal

### Features

#### Player Systems
- [ ] Расширенные статы (HP, Mana, Credits)
- [ ] Полноценный инвентарь с категориями
- [ ] Экипировка
- [ ] Уровни и опыт (опционально)

#### Inventory UI
- [ ] Панель инвентаря
- [ ] Drag & drop (опционально)
- [ ] Описания предметов
- [ ] Использование предметов

#### New Mechanics
- [ ] Торговля с NPC
- [ ] Крафтинг (опционально)
- [ ] Случайные события

#### Content
- [ ] Новые квесты с использованием инвентаря
- [ ] Предметы: ключи, инструменты, ресурсы
